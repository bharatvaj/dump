# Calculator
**Copyright: 2016, Ulōka**  
This calculator application was inspired by the material design calculator in Android, but with more advanced features  
Last succesful compilation with jdk version : **1.8.0_66** 

# How to use
Type in and get answers, that simple  
**NOTE**  
This program evaluates numbers in the given priority if you forgot to add brackets correctly, the regular BODMAS
 * Brackets
 * Exponents(Powers, Roots)
 * Multiply or Divide
 * Add or Subtract
